        form tag의 속성
        1. action : url(전송할 데이터를 받을 url) - 처리하는 views의 함수까지 request파라미터로 전달
        2. method : http method(get/post) -- default값 : get

                Mime type : 데이터에 대한 정보(데이터의 종류, 형식 표시) == 메타데이터이다.
                ex) text/css : 이 데이터는 문자 데이터이고 css 포맷
