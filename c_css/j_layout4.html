flex container

        container는 정렬할 요소들을 감싸는 부모요소
        display:flex/inline-flex 지정을 통해 container를 만들 수 있다.
        container의 자식요소는 item이 된다

        display:flex : block요소인 컨테이너
        display:inline-flex : inline요소인 컨테이너

        flex-direction : item이 정렬되는 방향
            row : 왼-오
            row-reverse : 오-왼
            column : 위-아래
            column-reverse : 아래-위
