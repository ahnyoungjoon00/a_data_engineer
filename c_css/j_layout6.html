        justify-content:주축에 대한 정렬방식을 설정
            flex-start:default, item을 시작점부터 정렬
            flex-end:item을 끝점부터 정렬
            center:가운데 정렬
            space-around:item을 일정한 간격으로 정렬
            space-between:일정한 간격으로 정렬, 시작/끝 item를 컨테이너의 시작/끝 지점에 배치
