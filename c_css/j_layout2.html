position:element가 배치되는 위치를 지정하는 속성

        기본값:static
            부모요소의 좌측 상단부터 순차적으로 배치
            순차적: 
                inline > 라인에 공간이 남아있으면 마지막 요소의 우측에 배치, 없으면 다음 줄에 배치
                block  > 다음 줄에 배치
            top, bottom, left, right, z-index 속성을 사용할 수 없다.
    
        relative:static과 대부분 동일
            부모요소의 좌측 상단부터 순차적으로 배치
            순차적: 
                inline > 라인에 공간이 남아있으면 마지막 요소의 우측에 배치, 없으면 다음 줄에 배치
                block  > 다음 줄에 배치
            top, bottom, left, right, z-index 속성을 사용할 수 있다.
    
        absolute:normalflow가 끝난 좌측 상단에 배치
                 top, bottom, left, right, z-index 속성을 사용할 수 있다.
        fixed:화면의 좌측 상단을 기준으로 요소가 배치
              top, bottom, left, right, z-index 속성을 사용할 수 있다.

top/bottom/left/right

        top:지정한 크기만큼 요소를 위에서 아래로 이동
        bottom:지정한 크기만큼 요소를 아래에서 위로 이동
        left:지정한 크기만큼 요소를 왼쪽에서 오른쪽으로 이동
        right:지정한 크기만큼 요소를 오른쪽에서 왼쪽으로 이동

        position:relative : 요소의 현재 위치를 기준으로 이동
        position:absolute : 부모요소의 top/bottom/left/right를 기준으로 이동
